# installation
```bash 
composer require nishadil/artisanpro
```


# Commands

### make blade file/view file
```bash
php artisan make:view nishadilView
```

OR

```bash
php artisan make:blade nishadilView
```


### create trait class
```bash
php artisan make:trait nishadilTrait
```
